🚀 TaskFlow — Team Task ManagerEk simple aur powerful full-stack web app jo teams ko projects aur tasks manage karne mein help karti hai. Ismein Role-Based Access Control (RBAC) hai, matlab Admin aur Members ki apni-apni powers hain.🌐 Live LinksLive URL: (https://web-production-99dd0.up.railway.app ) Repository: (https://github.com/Prakhar07012005/Task-Manager)🤔 Key Features (Isme kya-kya hai?)Authentication: Signup aur Login ka feature hai jahan user choose kar sakta hai ki wo Admin hai ya Member.Project Management: Admins naye projects bana sakte hain aur team members ko assign kar sakte hain.Task Tracking: Tasks create karna, priority set karna, aur deadlines track karna bohot easy hai.Status Flow: Tasks simple flow follow karte hain: To Do ➔ In Progress ➔ Done.Role-Based Access (RBAC): * Admins: Poore project aur sabhi tasks ko control kar sakte hain.Members: Sirf apne assigned tasks ka status update kar sakte hain.Dashboard: Ek personalized view jahan projects, tasks aur overdue items ek saath dikhte hain.🛠️ Tech Stack (Maine kya use kiya)LayerTechnologyBackendDjango 5.1 + Django REST FrameworkDatabasePostgreSQL (Production) / SQLite (Local)FrontendDjango Templates + Custom CSSAuthDjango Sessions + Custom User ModelDeploymentRailway + Gunicorn🔐 Permissions Table (Kaun kya kar sakta hai?)ActionAdminMemberSign up / Login✅✅Create Projects & Tasks✅❌Edit / Delete Everything✅❌Update Assigned Task Status✅✅View Full Team Progress✅❌⚙️ Local Machine Par Kaise Chalayein?Repo Clone Karein:Bashgit clone https://github.com/HarnoorXtiet/Task-Manager
